# Arcade-Cat

## Description

A rogue like arcade game that gives special abilities as the level progresses.

## Controls

- A / Left Arrow: left
- W / Space : Jump
- D / Right Arrow: Right
- Left Click: Shoot
- E: AOE Attack (REQUIRES UNLOCK FROM SHOP WHEN DEAD)
- Right Click: Block (REQUIRES UNLOCK FROM SHOP WHEN DEAD)
